**Description**

Simple skin-changing plugin for Velocity. Works on both pirated and licensed servers.
What the plugin does
With LiteSkins, any player can set themselves a skin from any licensed nickname — just by typing a command. It doesn't matter whether the server runs in offline (pirated) or online (licensed) mode: the skin is applied directly on the proxy, with no need to install anything on the backend servers.

**Features**
- /skin set <nickname> — set yourself the skin of a licensed account with the given nickname
- /skin clear — revert your skin back to default
- Skins are saved to a database so they don't need to be fetched again every time
- If a player is on a pirated account, you can set a default skin for them using the default-skins and default-skins-per-names options. As an example, default-skins comes with 9 sample skins: zuri, sunny, efe, etc.
- All plugin messages can be fully customized

**Requirements**
- Velocity 3.5.0 or higher
- Java 17 or higher
