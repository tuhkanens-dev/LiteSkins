**Description**
Simple skin-changing plugin for Velocity. Works on both pirated and licensed servers.
What the plugin does

LiteSkins lets players change their skin to match any Minecraft nickname, pulled directly from Mojang's servers — even on pirated (offline-mode) servers where players normally can't have custom skins. Everything happens on the proxy level, so no plugin installation is needed on the backend servers.

**Features**
- /skin set <nickname> — copy the skin from any existing Minecraft nickname onto yourself
- /skin clear — reset your skin back to default
- Skins are cached in a database so they don't need to be re-fetched every time
- For pirate accounts, you can assign default skins using the default-skins and default-skins-per-names options — comes with 9 sample skins out of the box (zuri, sunny, efe, etc.)
- All plugin messages are fully customizable

**Requirements**
- Velocity 3.5.0 or higher
- Java 17 or higher
